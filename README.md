# Pretty Git

Генератор красивых отчётов по Git-репозиторию: сводка, активность, авторы, файлы, тепловая карта.

- Лицензия: MIT.

## Требования
- Ruby >= 3.1 (рекомендуется 3.4.x)
- Git установлен в PATH

## Setup
```
bin/setup
```

## Тесты
```
bundle exec rspec
```

## Линтинг
```
bundle exec rubocop
```
